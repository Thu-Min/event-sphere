# Decisions

### Use pnpm as package manager

**Phase:** 1
**Context:** User preference for package manager across the project
**Decision:** Use pnpm instead of npm/yarn/bun
**Alternatives considered:**
- npm (create-next-app default)
- yarn
- bun

---

### Modern defaults for tooling

**Phase:** 1
**Context:** Scaffolding choices for a 2025/2026 Next.js project
**Decision:** App Router, Tailwind CSS, TypeScript strict, ESLint (Next.js defaults), Prettier (standard)
**Alternatives considered:**
- Pages Router (legacy)
- CSS Modules or styled-components
- Custom ESLint rules
